# 项目说明文挡

* npm install：安装所需要的模块

---

### gulp命令相关介绍

项目支持scss解析gulp-sass，html共用模板提取gulp-include，js模块化开发webpack(require),自动刷新browser-sync，数据代理http-proxy-middleware，发布时静态资源走CDN

* gulp dev:开启开发流程
* gulp build:构建发布资源（发布前先跑下gulp dev）

- - -
